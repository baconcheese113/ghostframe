# GhostFrame Frontend

Planned Next.js frontend for GhostFrame.

## Planned stack

- **Framework**: Next.js (App Router)
- **Styling**: Tailwind CSS
- **Language**: TypeScript

## Planned pages

- `/` — Landing page with project overview
- `/analyze` — Upload MAF/FASTA, configure analysis, view results
- `/results/:id` — Detailed results view with Sankey diagram (Silent → reclassified)
- `/demo` — Interactive demo with HPV16 sample data
